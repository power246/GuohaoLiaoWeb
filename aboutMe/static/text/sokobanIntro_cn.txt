简介
一个基于Python和pyphysicssandbox库开发的简易推箱子游戏，包含经典关卡设计。玩家通过方向键控制角色推动箱子到目标位置，目前仅实现单关卡玩法，未来计划扩展更多创意关卡。

功能特点
核心玩法
角色控制：使用W/A/S/D键移动角色（绿色球体）
箱子交互：推动蓝色箱子到黄色目标区域（碰撞检测逻辑）
胜利条件：所有箱子到达目标位置时显示“YOU WIN!”
物理引擎：通过pyphysicssandbox模拟物体移动和碰撞
关卡设计
地图系统：用坐标数组定义墙壁（红色方块）和目标点（黄色方块）
动态更新：箱子位置实时检测，避免重叠或穿墙
代码结构
模块化函数：box_move()处理移动逻辑，box_finish()检测胜利条件
全局状态管理：通过f/g等变量跟踪角色和箱子位置

技术栈
组件	用途
Python	核心开发语言
pyphysicssandbox	物理引擎（碰撞、移动模拟）
坐标系统	管理地图、角色和箱子的位置

挑战与解决方案
问题	解决方式
箱子碰撞逻辑	通过坐标校验防止箱子重叠或穿墙（h1==i or h1==h2等条件判断）
胜利条件检测	遍历所有箱子位置与目标点坐标匹配（h2==(300,200)等）
库依赖问题	提供pip install pyphysicssandbox安装指南

运行说明
安装依赖：
pip install pyphysicssandbox
启动游戏：
python @推箱子@.py
操作方式：
W/A/S/D键控制角色移动
将蓝色箱子推至黄色目标区域
