项目：Discord 音乐机器人
简介
本项目是一个基于 Discord 服务器的音乐机器人，支持播放 YouTube 音频，并提供多种播放模式。该机器人允许用户创建播放列表，并在 Discord 语音频道中播放音乐。

功能特点
音乐播放：支持从 YouTube 视频（包括直播）提取音频进行播放。
播放列表管理：用户可以添加、删除、查看播放列表，并按顺序或随机播放歌曲。
播放模式：支持单曲播放、列表播放、单曲循环和随机播放模式。
基本控制：提供加入/退出频道、播放、暂停、继续播放、调整音量等基础功能。
支持多语言：包含中文版本 (discord_bot.py) 和英文版本 (discord_bot_en.py)。

技术栈
后端：Python、Discord.py、yt-dlp（用于从 YouTube 获取音频）。
API 集成：使用 Discord API 进行命令控制，yt-dlp 进行音频处理。
部署（待定）：可以使用 Heroku、AWS、VPS 或本地服务器运行。

挑战与解决方案
令牌安全性：机器人 Token 已隐藏，需手动更换为自己的 Token 以运行程序。
播放模式管理：当前版本在播放过程中可能无法切换播放模式，建议在播放前设定模式。
YouTube 音频解析：使用 yt-dlp 解析音频流，并通过 FFmpeg 进行格式转换和播放。

未来改进方向
优化播放模式切换，允许用户在音乐播放过程中无缝切换模式。
添加更丰富的交互功能，如歌词显示、音乐搜索、自动推荐等。
支持更多音频源，如 Spotify、SoundCloud 等。
优化错误处理机制，提供更友好的错误提示和解决方案。
