项目：个人网站
简介
本项目是一个使用 Django 作为后端框架，并结合 HTML、CSS 和 JavaScript 进行前端开发的个人网站。其目标是打造一个个人作品集，以整洁、美观的方式展示我的经历、项目和技能。

功能特点
主页（Homepage）： 一个简洁的欢迎页面，介绍我的基本信息。
关于我（About Me）： 详细介绍我的背景、兴趣以及编程旅程。
项目（Projects）： 一个专门展示我所做过项目的页面，包括该网站本身。
经验（Experience）： 展示我在 VEX 机器人、编程以及其他技术领域的经验。

动态内容：
使用 Django 的 模板系统（{% block %}）实现网页结构的可复用性。
通过 .txt 文件 读取文本内容，使内容管理更加灵活，便于更新。
通过 自定义 CSS 让文本能够围绕图片动态排版。

交互元素：
站内超链接，方便用户在不同页面之间切换。
CSS 动画和悬停（hover）效果，提升视觉体验。
响应式设计（Responsive Design）： 确保网站在不同设备和屏幕尺寸下都能良好显示。

技术栈
后端（Backend）： Django（Python）
前端（Frontend）： HTML、CSS、JavaScript
样式（Styling）： CSS Flexbox、Grid，以及自定义排版和布局
部署（Hosting）：（待定，可选择 GitHub Pages、Heroku 或 AWS 等平台）

挑战与解决方案
动态文本格式化： 起初，从 .txt 文件中读取文本并保持格式是一个挑战。我使用 Python 的 .replace() 方法，将对应文本更换为可以被HTML解析的格式，从而正确换行，并动态处理超链接。
多页面排版继承： 我使用 Django 的模板继承机制，使多个网页共享统一的布局，提高可维护性和代码复用性。
URL 管理： 通过 Django 的 reverse() 方法动态生成 URL，而不是硬编码链接，使得网站更易维护。

未来改进方向
收集访问用户数据，通过数据分析解析每个用户在每个页面上停留的时间，用于优化内容。
扩展项目（Projects） 部分，增加更详细的描述和交互式内容。
通过优化 CSS 响应式设计 进一步提升网站的适配性。

这个项目不仅展示了我的技术能力，也体现了我对 Web 开发的热情。目前它仍在不断完善中，我期待在未来继续优化和扩展其功能！
